42.一个4000字节的IP数据包经过MTU值为1400字节的接口时，需要将其分成3片，请问分片后每个数据包大小分别是多少字节？(C)。 ​


A.1400/1400/1200
B.1400/1400/1240
C.1396/1396/1248
D.1500/1500/1000

解析：
  • 4000字节的长度包含20字节的首部长度，所以数据长度为3980字节
  • MTU是1400，去除20字节的首部长度，得1400 - 20 = 1380，但1380 / 8除不尽，所以取最近能整除8的1376为分片长度
  • 三个分片分别为
    • 0 - 1375，长度为1376
    • 1376 - 2751，长度为1376
    • 2752 - 3979，长度为1228
  • 三个分片分别加上首部长度20，为1396/1396/1248

  • 5E ⊕ 20 = 0111 1101 = 7E

19.海明码 ​

海明码是一种纠错码，对于32位的数据，至少需要增加(D)个校验位才能构成海明码。
A.3
B.4
C.5
D.6

11.CMDA举例。 ​

共4个站进行码分多址CDM通信，4个站的码元序列为：
A.(-1-1-1+1+1-1+1+1) B.(-1-1+1-1+1+1+1-1)
C.(-1+1-1+1+1+1-1-1) D.(-1+1-1-1-1-1+1-1)
接收站收到的码片序列为(-1+1-3+1-1-3+1+1)(各个站发送数据的和)，求哪个站发送了数据？发送了什么数据？
解析：

用码片序列和接收到的序列做规格化内积，1表1，-1标识0，0表示未发送
  • 站点A内积：[(−1)×(−1)]+[(−1)×1]+[(−1)×(−3)]+[(+1)×1]+[(+1)×(−1)]+[(−1)×(−3)]+[(+1)×1]+[(+1)×1] = 8
    • 8 / 8 = 1，说明站点 A 发送了数据 1
  • 站点B内积：[(−1)×(−1)]+[(−1)×1]+[(+1)×(−3)]+[(−1)×1]+[(+1)×(−1)]+[(+1)×(−3)]+[(+1)×1]+[(−1)×1] = -8
    • -8 / 8 = -1，说明站点 B 发送了数据 0
  • 站点C内积：[(−1)×(−1)]+[(+1)×1]+[(−1)×(−3)]+[(+1)×1]+[(+1)×(−1)]+[(+1)×(−3)]+[(−1)×1]+[(−1)×1] = 0
    • 0 / 8 = 0，说明站点 C 未发送数据
  • 站点D内积：[(−1)×(−1)]+[(+1)×1]+[(−1)×(−3)]+[(−1)×1]+[(−1)×(−1)]+[(-1)×(−3)]+[(+1)×1]+[(−1)×1] = 8
    • 8 / 8 = 1，说明站点 D 发送了数据 1

答：站点A，站点B，站点D发送了数据，站点A发送了数据1，站点B发送了数据0，站点D发送了数据

4.信道宽度为4KHz，信噪比为30dB，根据香农定理，信道的最大传输速率约为多少？。 ​


解析：
根据信噪比公式
信噪比(dB) = 10log10(S/N)(dB)

得：30dB = 10 x log10 (S/N)
因此：S/N = 1000
由香农定理
C = Wlog2(1 + S/N)

得 C = 4000 x log2(1 + 1000) ≈ 40000bps = 40kbps
